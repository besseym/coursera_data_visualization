Background

This is the first assignment in the Coursera Course on Data Visualization. This class is one of several required classes in the Data Mining Specialization.

The goal of this assignment is to gain some experience with handling and deciding how to visualize some data and to get a feel for the various aspects of the visualization.

This assignment will also help me analyze other visualizations and assess their effectiveness.

Data

The data visualized in the following charts came from the NASA's Goddard Institute for Space Studies (GISS). The data consists of world surface temperatures updated every month using current data files from NOAA GHCN v3 (meteorological stations), ERSST v4 (ocean areas), and SCAR (Antarctic stations).
http://data.giss.nasa.gov/gistemp/

Specifically the visualized data came from Table Data: Global and Hemispheric Monthly Means and Zonal Annual Means under Combined Land-Surface Air and Sea-Surface Water Temperature Anomalies (Land-Ocean Temperature Index, LOTI). This table data consists of plain text files in tabular format of temperature anomalies, i.e. deviations from the corresponding 1951-1980 means.
