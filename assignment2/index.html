This is a Network Graph Visualization of my personal website http://www.michaelbessey.com/.

A node's color indicates the section it belongs to within the website. The color black indicates an external link. The size of each node is based on how many other nodes are connected to it. One can click on a node to reveal a list of its connected nodes. Then one can mouseover that list to highlight the connection between that node and the selected node. Finally, you can click on a node in the list to make it the selected node.
