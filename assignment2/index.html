This is a Network Graph Visualization of my personal website http://www.michaelbessey.com/. The size of each node is based on how many other nodes are connected to it. One can click on a node to reveal a list of its connected nodes. Then one can mouseover that list to highlight the connection between that node and the selected node. Finally, you can click on a node in the list to make it the selected node.
